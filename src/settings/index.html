Blocking, timers, and distraction control — the tools that keep you on task.

Focus Mode

Hides the Group button and (optionally) nudges you when you've opened too many tabs. Doesn't block anything on its own — pair it with Site Blocking for that.

A gentle nudge only, not a hard block — you'll still be able to open more tabs.

Pomodoro Timer

Changes apply immediately. A running timer keeps its current countdown; only the next phase picks up new durations.

Turns both on when a work session starts, then restores whatever you had before — not just "off" — when the break begins.
